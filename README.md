# Blu Gas Website

เว็บไซต์ของบริษัทบลูแกส สร้างด้วย React

## การติดตั้ง

```bash
npm install
```

## การรันโปรเจกต์

```bash
npm start
```

เว็บไซต์จะเปิดที่ http://localhost:3000

## การ Build สำหรับ Production

```bash
npm run build
```

## โครงสร้างโปรเจกต์

- `src/components/` - คอมโพเนนต์ต่างๆ ของเว็บไซต์
- `public/` - ไฟล์สาธารณะ
- `Banner.png` - ภาพ Banner สำหรับหน้าแรก

## หน้าเว็บไซต์

- หน้าแรก (Hero Banner)
- เกี่ยวกับเรา
- ผลิตภัณฑ์และบริการ
- ผลงานของเรา
- ข่าวสารองค์กร
- ติดต่อเรา
